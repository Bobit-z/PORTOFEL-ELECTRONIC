# PORTOFEL-ELECTRONIC

Aplicatie web demo pentru portofel electronic.

## Pornire locala

```powershell
python app.py
```

Deschide in browser:

```text
http://127.0.0.1:5000/
```

```text
[http://127.0.0.1:5000/](https://portofel-electronic.onrender.com/login)/
```

## Deploy

Pentru hosting online, foloseste comanda:

```bash
python app.py
```
